[台北市/北投區]靠近溫泉街的陳季炸雞

老實說這家店是因為之前騎車亂晃無意間看到的，因為賣的炸物和價格與石牌一間炸物店一模一樣因此吸引了我的注意，而且又在溫泉街附近，如果之後有來泡溫泉晚上宵夜就多了一個選擇。地點還算是滿好找的，就在公館路路口附近，配上大大的橘黃色看板應該不至於找不到。

看板上面寫了有在賣的炸物，特別是那個雞翅七支100，只要把中間的店名換掉根本就是石牌另一間炸物店，基本上店員不太理人所以很容易被插隊，可能你準備要點餐時旁邊就突然插一個進來然後你就會被放置PLAY。這次為了和石牌的炸雞做比較所以同樣點了七支雞翅，先從顏色上來說並不是那麼的漂亮感覺看起來有點像是丹丹那樣黑黑的，裹粉吃起來很普通沒什麼味道，最奇怪的是雞肉吃起來也沒什麼味道，如果沒有加胡椒鹽和辣椒粉感覺吃起來和水煮雞一樣什麼味道都沒有。

總結來說沒有特別好吃但也不到難吃，但是讓我沒有再去買第二次的動力，下次找時間再來介紹石牌好吃的炸物店！
